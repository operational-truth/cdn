Typography #

System fonts dominate for fast loading with monospace for code. Headings use tight line-height; body copy stays at 15px with 1.6 leading.

Page Title (32px / 700)

Section Heading (22px / 600)

Subsection (18px / 600)

Body text paragraph (15px / normal)

Small text for captions (13px)

Inline code uses the code element with a subtle rose hue to stand out.

Spacing #

The spacing scale is based on a 4px baseline: tiny gaps for keys, 16px for default padding, 32-48px for major sections.

Token Value Usage
spacing-xs 4px Keyboard keys, micro gaps
spacing-sm 8px Icon/text gaps, compact elements
spacing-md 16px Default padding, section gaps
spacing-lg 24px Card padding, component margins
spacing-xl 40px Page-level margins
