HTML-first table upgrade (preferred)

We treat the HTML table as the authoritative source of truth and let natural-grid enhance it. This keeps the markup readable, accessible, and easy to edit without synchronizing JSON snapshots. JSON configs can still be helpful for quick prototypes, but for production content we first author the table and then ask the grid to upgrade it via data-nwua-init="table" with the table as the first child.
