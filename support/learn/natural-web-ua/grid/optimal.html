Optimal table upgrade flow

The grid can enhance an existing HTML table without the page needing to reference <natural-grid>. Call naturalGridUpgrade(id) for a specific table or let upgradeToNaturalGrids() automatically convert every table[data-nwua-grid] on the page after load.

Manual upgrade (explicit call)

Assign an id to the table and call naturalGridUpgrade("manual-table") once the DOM is ready. This is great when you only have one legacy table or need to control the timing of the transformation.

Declarative upgrade (selector driven)

Tag any table with data-nwua-grid and optionally mix in attributes like data-nwua-grid-theme="modern". upgradeToNaturalGrids() finds these tables, reads the metadata, and replaces them with Natural instances after load. This keeps your HTML markup simple and future-proof.
