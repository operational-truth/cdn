Introduce natural-web-ua/elements.js helpers

While the module exports every tag as a callable function (e.g., div(), section(), ul()), the real benefit comes when you combine helpers like each(), classNames(), and styleText() to keep your DOM creation deterministic and composable.

Here is the code that the module script below mirrors:

import * as h from "../../../../natural-web-ua/elements.js";

const features = [
  { title: "Composable tags", detail: "Use native HTML elements as functions." },
  { title: "Helpers", detail: "classNames(), attrs(), styleText(), each() keep markup tidy." },
];

const panel = h.section(
  { class: "demo-panel" },
  h.h3("Rendered with natural-web-ua/elements.js"),
  h.ul(
    { class: "feature-grid" },
    h.each(features, (feature, index) =>
      h.li(
        { class: h.classNames("feature-card", { "feature-card--even": index % 2 === 0 }) },
        h.h4(`${index + 1}. ${feature.title}`),
        h.p(feature.detail),
      ),
    ),
  ),
);
